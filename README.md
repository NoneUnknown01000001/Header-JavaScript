# Header-JavaScript

A lightweight JavaScript utility for dynamically creating and managing headers in web applications.  
This project is designed to simplify header manipulation, making it easier to add, update, and style headers without repetitive code.

---

## 🚀 Features
- Create headers dynamically with JavaScript
- Easily update header text and styles
- Lightweight and dependency-free
- Simple integration into any HTML project
